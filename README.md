## 7.2 分治

- **分治法的步骤如下**：

1. **分**：将问题分解为规模更小的子问题
2. **治**：将这些规模更小的子问题逐个击破
3. **合**：将已解决的子问题合并，并最终得出母问题的解

- **分治与递归的联系与区别**：

1. 两者都体现了将问题分解成更小子问题的思想；
2. 递归强调的是可以从子问题逐步推出母问题；
3. 分治强调的是母问题是由多个子问题合并之后解决。
